***entity包***

​	该包为数据对象包,主要放置实体类(entity),简单java对象(pojo)等数据对象.在开发中请遵循以下约定:

* 实体类请实现序列化接口
* 实体类属性命名请严格按照驼峰命名法
* 属性名请用意义完整的英文描述,应尽量避免使用汉语拼音
* 请为每个实体类重载toString()方法
* 请在属性名右侧添加单行注释,进行简单描述
